## 首次使用配置

### 1. 注册账号

1. 访问应用首页，点击"立即注册"
2. 填写：
   - 用户名
   - 邮箱地址
   - 密码（至少6个字符）
3. 点击"注册"按钮
4. 自动跳转到主页面

### 2. 配置 API 密钥

**重要**：为了使用 AI 功能，您需要配置阿里云百炼平台的 API Key。

1. 点击导航栏的"设置"
2. 在"API 密钥配置"区域：
   - **阿里云百炼平台 API Key** (必填)
     - 获取地址：https://bailian.console.aliyun.com/
     - 格式：`sk-xxxxxxxxxxxxxxxx`
   - **高德地图 API Key** (可选)
     - 获取地址：https://console.amap.com/dev/key/app
     - 用于地图显示功能
3. 点击"保存设置"

## 创建旅行计划

### 方式一：语音输入（推荐）

1. 进入"创建行程"页面
2. 点击"🎤 开始语音输入"按钮
3. 说出您的旅行需求，例如：
   ```
   "我想去日本，5天，预算1万元，喜欢美食和动漫，带孩子"
   ```
4. 系统会自动识别并填充表单
5. 检查自动填充的信息，根据需要调整
6. 点击"✨ 生成旅行计划"

### 方式二：手动输入

1. 进入"创建行程"页面
2. 填写表单：
   - **目的地**：例如"日本东京"
   - **开始日期**：选择出发日期
   - **结束日期**：选择返回日期
   - **预算**：输入预算金额（元）
   - **人数**：选择同行人数
   - **旅行偏好**：点击选择您的偏好标签
3. 点击"✨ 生成旅行计划"

### 生成过程

- AI 会根据您的需求生成详细的旅行计划
- 包括：
  - 每日行程安排
  - 景点推荐
  - 餐厅建议
  - 住宿推荐
  - 交通方式
  - 费用预算
- 生成通常需要 10-30 秒

## 管理行程

### 查看行程列表

1. 点击"我的行程"
2. 查看所有已创建的旅行计划
3. 每个卡片显示：
   - 行程标题
   - 目的地
   - 日期
   - 人数
   - 预算和预估花费

### 查看行程详情

1. 在行程列表中点击"查看详情"
2. 查看完整的日程安排：
   - 每日活动
   - 时间安排
   - 地点信息
   - 费用明细

### 编辑行程

1. 在行程详情页点击"编辑"
2. 修改行程信息
3. 点击"保存"

### 删除行程

1. 在行程卡片上点击"删除"
2. 确认删除操作
3. 行程将永久删除

## 费用管理

### 添加开销记录

#### 方式一：语音记录
1. 进入费用管理页面
2. 点击"🎤 语音记录"
3. 说出开销信息，例如：
   ```
   "午餐花了150元"
   "打车到景点20元"
   ```

#### 方式二：手动添加
1. 点击"添加开销"
2. 填写：
   - 类别（住宿/餐饮/交通等）
   - 金额
   - 描述
   - 日期
3. 点击"保存"

### 查看费用统计

- 查看总开销
- 按类别统计
- 与预算对比
- 图表可视化

## 常见问题

### Q: 语音识别不工作怎么办？

**A**: 
1. 确保使用 Chrome 或 Edge 浏览器
2. 允许浏览器访问麦克风权限
3. 必须在 HTTPS 或 localhost 环境下使用
4. 检查麦克风设备是否正常

### Q: AI 生成失败怎么办？

**A**:
1. 检查 API Key 是否正确配置
2. 确认阿里云账户余额充足
3. 尝试简化需求重新生成
4. 查看浏览器控制台错误信息

### Q: 无法登录怎么办？

**A**:
1. 检查邮箱和密码是否正确
2. 确认已完成注册
3. 检查网络连接
4. 清除浏览器缓存后重试

### Q: 数据没有同步怎么办？

**A**:
1. 检查是否已登录
2. 确认网络连接正常
3. 刷新页面重试
4. 检查 Supabase 配置是否正确

### Q: 地图不显示怎么办？

**A**:
1. 检查高德地图 API Key 是否配置
2. 确认网络可以访问高德地图服务
3. 刷新页面重试

### Q: 如何导出行程？

**A**:
目前支持在浏览器中查看和打印。未来版本将支持：
- PDF 导出
- 分享链接
- 日历同步

## 最佳实践建议

### 1. 语音输入技巧

**清晰描述**：
- ✅ 好："我想去日本东京，5天，预算一万元"
- ❌ 差："去日本玩玩"

**包含关键信息**：
- 目的地
- 天数或日期
- 预算
- 人数
- 特殊偏好

### 2. 预算设置建议

建议预算包含：
- 往返交通费用
- 住宿费用
- 餐饮费用
- 景点门票
- 其他开销
- 10-20% 的预留金

### 3. 行程规划建议

- 不要安排过满，留出休息时间
- 考虑交通时间
- 注意景点开放时间
- 预留机动时间应对突发情况

### 4. 费用记录建议

- 及时记录每笔开销
- 保留电子或纸质收据
- 分类记录便于统计
- 定期查看预算使用情况
